## B 创始人背景 → 研发思路与商业模式 的影响分析【核心】

### 核心判断 / Core thesis

Earendil / 华深智药不是单一“AI工具服务公司”，而是“AI founder-led + antibody development veteran co-led”的双核心架构：彭健定义底层平台与生成/预测能力，朱祯平把平台压进真实抗体管线、临床转化和BD交易。该组合直接决定了公司从一开始就更像“AI-native biologics asset company”，而不是传统CRO或SaaS。

### 背景对战略的影响 / Founder background -> strategy

1. 平台优先但不止于平台 / Platform-first, but not platform-only  
   彭健的计算生物学背景使公司早期叙事围绕 OmegaFold、蛋白结构预测、高通量实验闭环和AI生成大分子。2022年A轮材料明确称公司要打造“下一代人工智能科学计算平台”，结合自研高通量生物实验技术，并推进自主研发与对外合作。来源：高榕资本转载/阿里云创新中心，2022-06-22：https://startup.aliyun.com/info/1033953.html

2. 朱祯平加入后，从平台验证转向资产化 / After Zhu joined, platform validation shifted toward assetization  
   2023年华深智药新闻明确写到朱祯平将负责创新药物研发、产品管线推进、战略制定与运营，并称其加入后将“加速建立内部药物研发团队和自研管线”。这说明公司战略从“平台被药企评估认可”转向“平台+内部药物研发组织”。来源：华深智药新闻，2023-02-10。

3. 靶点和模态选择体现药物开发老兵的商业纪律 / Target and modality choices show pharma discipline  
   Sanofi授权的 HXN-1002/HXN-1003 并非冒险探索完全未知靶点，而是在 α4β7、TL1A、IL23 等已被临床/商业验证的免疫通路上做双抗组合和工程优化。投资含义：公司并不是“用AI赌全新生物学”，而是用AI扩大复杂双抗设计空间，在已有免疫学路径上创造更优分子格式。来源：Earendil-Sanofi授权新闻，2025-04-17：https://www.prnewswire.com/news-releases/earendil-labs-announces-worldwide-exclusive-license-agreement-with-sanofi-for-next-generation-bispecific-antibodies-for-autoimmune-and-inflammatory-bowel-diseases-302431020.html；Sanofi 2025半年度报告，2025年披露：https://www.sanofi.com/assets/dotcom/content-app/publications/half-year-financial-reports/Half-year-financial-report-2025.pdf

4. 与同赛道差异 / Difference versus peer founder archetypes  
   - BigHat、LabGenius 类：更强调AI/自动化闭环和多参数优化，本身也走合作+自研，但公开临床资产验证较弱。  
   - Absci 类：上市公司、AI药物创建叙事强，但资产优先级会受资本配置影响，已有项目战略降级案例。  
   - AbCellera 类：生物实验/单细胞发现/制造一体化更强，AI是组合能力之一而非唯一核心。  
   - Earendil / 华深智药：更像“计算创始人+抗体药企老兵”驱动的双抗资产公司，BD节奏和临床推进速度比纯平台公司更激进，但平台真实边际贡献更需要数据室验证。

### 投资策略结论 / Investment implication

该团队由AI背景主导起步，但药物开发背景决定了商业化落点。彭健使公司有“新可能性”的底层叙事，即在MSA不足、双抗组合空间巨大、复杂生物药可开发性难以枚举时，用AI生成/筛选过去难以系统搜索的分子；朱祯平使公司避免停留在算法展示，转向可授权、可临床、可CMC放大的资产组合。投资判断应重点看：AI是否真的提高双抗PCC质量、减少实验轮次、形成负数据飞轮，而不是只看Sanofi headline deal value。

### VC判断 / VC read

Earendil 当前最可投资的不是纯软件收入，而是“AI平台产生可BD资产”的价值捕获。Sanofi交易证明大药企愿意为其早期双抗资产和平台合作支付确定性现金，但headline总额绝大多数仍是条件性里程碑，不应等同于实际收入。商业风格偏激进资产化：融资、BD、临床和CRDMO合作均服务于快速扩大多资产管线。

### 叙事是否漂移 / Narrative drift

不是反复更换故事，而是连续升级：  
1. 2021-2022：AI大分子建模/设计平台 + 高通量实验闭环。  
2. 2023：引入抗体药物开发老兵，建立内部研发团队和自研管线。  
3. 2025：双抗资产授权给Sanofi，证明平台可以产出可交易资产。  
4. 2026：平台级合作、超大额融资、临床推进、WuXi CMC合作，多资产公司化。

需要警惕的不是“叙事漂移”，而是“叙事膨胀”：公司从平台验证到40+项目、多个IND计划和$787m融资，节奏很快；数据室必须核实每个项目的PCC标准、实验轮次、失败率、负数据、CMC指标和IP归属。
