# 2.2 抗体亲和力 特异性和可开发性优化 — 代表公司深度研究卡

最近核验日期：2026-06-23

代表公司：Earendil Labs / 华深智药 / 关联公司 Helixon Therapeutics、Helixon Biotechnology (Suzhou) Co., Ltd.

## 0. 选择理由（为何这家代表本细分）

本轮自动选择 Earendil Labs / 华深智药，原因是其在“AI抗体/蛋白设计平台 + 高通量实验验证 + 自研临床资产 + 大药企BD现金验证”四个维度的组合强于第一轮名单中的其他候选。

- 与 BigHat、LabGenius 相比：BigHat/LabGenius的闭环平台证据较清楚，但公开资产仍主要停留在临床前或合作序列未披露；Earendil 已披露 HXN-1001 进入 IIa 首例给药（公司公告，C级），并有 Helixon Biotechnology (Suzhou) 的 HXN6005 I期注册（ClinicalTrials.gov，A级）。
- 与 Absci 相比：Absci的公开上市公司文件和临床注册证据强，但 ABS-101 已被公司战略降级为寻求合作伙伴推进，且BD现金验证不如 Earendil 与 Sanofi 两次交易集中。
- 与 AbCellera 相比：AbCellera临床、制造与商业化最成熟，但AI只是其一体化抗体发现平台的一部分；本细分要求聚焦“亲和力、特异性、可开发性优化”，Earendil 对“预测蛋白建模 + 高通量实验验证 + 双抗/多抗设计”的叙事和交易资产更贴近该技术细分。
- 最强外部验证：Sanofi 2025 Form 20-F 确认 2025-04 授权 HXN-1002/HXN-1003，首付款 1.25亿美元，最高开发/商业里程碑 17亿美元，销售分级提成（A级）；公司公告另披露 2026-01 第二次 Sanofi 平台合作，最高潜在总额 25.6亿美元，含最高 1.6亿美元 upfront/近端付款（C级）。

结论：Earendil 是本细分当前最具代表性的“AI驱动抗体优化/双抗设计已转化为临床资产与大药企BD”的案例，但平台真实闭环、负数据规模、AI对分子差异化的边际贡献仍需进一步尽调。

## 11. 本技术细分价值定性【降本增效 vs 范式重构】

结论：兼有；当前阶段为“产业验证期，局部进入初步商业化”。

### A. 降本增效部分

抗体亲和力、特异性和可开发性优化本质上仍嵌入传统抗体药研发流程：靶点选择、命中发现、亲和力成熟、功能筛选、可开发性评估、CMC、IND、临床。AI没有取消这些环节，但可以提高早期候选质量和资本效率。

具体替代/增强的环节：

- 替代部分人工经验式CDR突变、构型筛选、人源化和可开发性预判。
- 减少需要表达、纯化和测试的候选物数量，把湿实验资源集中到更可能满足多属性TPP的序列/构型。
- 提前识别表达差、聚集、热稳定性不足、高黏度、免疫原性、脱靶或双抗构型不合理等风险。
- 对双抗/多抗尤其有价值，因为传统组合空间过大，手工枚举和低通量试错效率低。

证据强度：Earendil、BigHat、LabGenius、Absci等公司均声称有AI+湿实验闭环；LabGenius有较清楚的论文级闭环案例；Earendil有Sanofi交易和临床资产外部验证。但整个细分仍缺少跨公司、可重复的量化证据证明“AI显著提高临床成功率”。

### B. 范式重构部分

本细分的“范式重构”主要发生在复杂抗体格式和多目标优化上，而不是普通单抗的单点亲和力成熟。

传统方法的难点：

- 双抗/多抗需要同时搜索两个或多个结合臂的亲和力、表位、拓扑、linker、价态、空间几何、表达和稳定性。
- 肿瘤/正常组织选择性、条件性活化、pH依赖性、低免疫原性和高浓度皮下注射等目标彼此牵制。
- 可行设计空间远大于可实验测试空间，传统方法难以在合理时间和成本内系统探索。

AI新增能力：

- 结构/序列联合建模可把大量不可行构型提前剔除。
- 生成模型和主动学习可提出非直觉序列/构型，并通过高通量实验回流修正。
- 多属性优化可把亲和力、特异性、功能、表达、稳定性、制剂和CMC风险放入同一搜索框架。

已实现程度：

- 已实现：平台公司能把AI设计资产推到授权和早期临床；Earendil-Sanofi是强商业验证，ClinicalTrials.gov也出现Helixon临床资产。
- 未完全实现：尚未有足够临床疗效/安全性读出证明AI优化抗体在患者层面优于传统抗体；公开数据也不足以证明候选物数量、周期、成本和成功率的系统性改善。

### 最终定性

本细分不是“AI完全取代抗体研发”，而是“在抗体/双抗设计空间急剧扩张后，AI+高通量实验使多属性优化更可规模化”。短期3-5年，确定性商业价值主要来自降本增效和BD授权；范式重构价值集中在双抗、多抗、条件性抗体、复杂制剂和跨靶点平台化设计，仍处产业验证期。

一句话总结：Earendil Labs / 华深智药 — 最高验证级别 A级（Sanofi 20-F、ClinicalTrials.gov）— 投资定位：持续跟踪、接近重点投资候选 — 本细分价值定性：兼有，当前为产业验证期、局部初步商业化。
