# 4.3 靶点-疾病-药物关系建模 — 代表公司深度研究卡

最近核验日期：2026-06-23  
代表公司：Insilico Medicine（英矽智能，HKEX: 03696.HK）  
选择方式：从第一轮候选公司中按“平台相关性 + 实验闭环 + 自研/合作资产 + 临床/监管/BD验证”综合排序，自动选取 Insilico Medicine。

## 0. 选择理由（为何这家代表本细分）

Insilico Medicine 是本细分最具代表性的样本：其 PandaOmics 明确用于靶点-疾病关系建模与靶点优先级排序，Chemistry42 进一步连接到分子生成，形成“靶点-疾病-药物”从假设到候选物的连续链条。与 Verge、CytoReason、BenevolentAI、BioMap 相比，Insilico 同时具备：

- 平台证据：PandaOmics 用 omics、文献、临床试验、专利等多源数据做 actionable target identification，并输出基因-疾病关联报告、靶点评估、药物再利用/候选优先级。[PandaOmics](https://pharma.ai/pandaomics)【C/E】
- 实验与临床验证：TNIK/IPF 项目 Rentosertib（ISM001-055/INS018_055）已完成中国 Phase IIa 并发表于 Nature Medicine；另有美国 Phase IIa 注册试验。[Nature Medicine](https://www.nature.com/articles/s41591-025-03743-2)【B】、[NCT05975983](https://clinicaltrials.gov/study/NCT05975983)【A】
- 资产转化：公司披露 40+ 项目、31 个 2021 年以来 PCC、13 条管线获 IND 批准/许可，最高到 Phase II。[Pipeline](https://insilico.com/pipeline)【C】
- 商业化验证：Exelixis 以 8000 万美元首付款获得 ISM3091 全球权益；2026 年 Lilly 合作含 1.15 亿美元 upfront、总额约 27.5 亿美元；SK Biopharmaceuticals 合作总额超过 25 亿美元但 upfront/近期里程碑仅 1800 万美元。[Exelixis](https://ir.exelixis.com/news-releases/news-release-details/exelixis-and-insilico-medicine-enter-exclusive-global-license)【A】、[Lilly](https://insilico.com/news/uiy12zcjg1-insilico-medicine-announces-global-rampd)【C】、[SK](https://www.prnewswire.com/news-releases/insilico-medicine-and-sk-biopharmaceuticals-achieved-ai-powered-drug-discovery-collaboration-worth-up-to-2-5-billion-for-neuroimmune-disorders-302806072.html)【C/D】

结论：Insilico 不是纯“关系建模软件商”，而是把关系建模嵌入 AI-native biotech 资产开发和 BD 的综合样本，最适合作为本技术细分的代表公司。

技术评价：

- 输入广：组学、知识图谱/文本、临床试验、专利、化合物/生物药、内部实验数据。
- 任务链完整：靶点发现 → 靶点评估 → 分子生成 → 实验验证 → 临床开发判断。
- 输出可操作：不是只生成“相关性图谱”，而是可直接进入 hit/lead/PCC/IND 的项目包。

- 可复用性强：已跨 fibrosis、oncology、immunology、CNS、metabolic/pain 等方向复用；但不同疾病中靶点发现可靠性仍需逐个项目验证。

## 4. 数据壁垒

数据来源：

- 公共与第三方数据：GEO/多组学、文献、专利、临床试验、grant、药物/生物药数据库等。
- 合作方数据：药企合作、软件客户上传或合作项目数据，是否可回流训练取决于合同，未完全披露。
- 内部实验数据：自动化实验室、内部管线、体内外模型、DMPK/安全性、临床数据。公司 2026 年强调“数据 + foundation models + agent systems”的持续集成。

壁垒判断：

- 数据规模壁垒：中高。PandaOmics 披露的数据库规模大，但其中大量来自公共/可购买/可爬取来源，绝对独占性有限。
- 负数据壁垒：未充分披露。AI 药物发现真正稀缺的是失败实验、不可成药靶点、不可优化分子、临床不转化数据；公司没有系统披露负数据覆盖率、可用性和回流机制。
- 闭环数据壁垒：较高但仍需验证。内部 31 个 PCC、13 个 IND、10 个临床项目若能形成统一可训练数据资产，将显著强于纯软件商；但外部无法验证数据治理、标签质量与合同可用性。
- 商业数据壁垒：中高。13 家 top 20 全球药企使用软件和多项合作带来 workflow 嵌入，增强 switching cost；但药企通常会限制敏感数据回流。
- 最大不确定性：公司创始人公开表达过“大量数据本身未必是竞争优势”的观点；更核心的壁垒可能是模型、实验执行、项目选择和 BD 能力的组合，而不是单一数据库。

## 8. 反面信息与风险

已检出/需重点跟踪的风险：

1. 临床证据仍早期：Rentosertib Phase IIa 为 71 例、12 周，FVC 信号积极但样本量小、周期短；Nature Medicine 论文也提示需要更大规模、更长周期验证。尚无 Phase III 或注册性终点。【B】
2. AI 归因风险：Rentosertib 的成功来自 AI + 人类专家 + 药化/DMPK + 动物/临床执行的整体系统，不能简单归因于“AI 自动发现”。PandaOmics 提出和排序假设，但靶点验证和分子优化仍需大量传统药物发现工作。
3. BD headline 夸大风险：US$1.2B、US$2.5B、US$2.75B 多为里程碑总额；实际现金通常为 upfront、near-term milestones 和已触发 milestones。SK 案例尤其明显，最高 US$2.5B 但近期现金 up to US$18M。
4. 数据壁垒可复制风险：大量输入来自公共组学、文献、专利、临床试验库；真正独占的是内部实验/临床回流和 workflow，而非公共数据库本身。
5. 业务复杂度风险：公司同时做软件、平台合作、自研管线、资产授权、自动化实验室、foundation models、材料/农业/营养/兽药拓展，存在资源分散和叙事过宽风险。
6. 资本市场风险：2025 年上市后拥有现金，但 2025 年仍亏损。公司年度结果披露 2025 收入 US$56.24M，现金 US$393.3M；第三方/年报摘要显示净亏损较大，商业收入尚不足覆盖持续临床投入。
7. 合作执行风险：Lilly/SK/Menarini/Exelixis 后续是否推进到关键临床读数、是否继续付款，取决于合作方优先级和临床数据。

未检出强负面：

- 截至 2026-06-23，未检出 Insilico 大规模裁员、重大监管处罚、核心临床失败公告或重大合作退出的可靠一手证据。需持续监控 layoffs/discontinued/terminated/pivot/controversy 关键词。

## 9. 投资判断

投资定位：重点投资 / 持续跟踪之间，偏“重点投资级案例”，但上市后应按公开市场估值纪律进入，不适合仅凭 AI 叙事追高。

判断：

- 问题重要性：高。靶点选择错误是药物研发高失败率的核心原因之一，本细分直接作用于最早期、最高杠杆的决策点。
- AI 是否不可替代：部分不可替代。多组学 + 文献 + 临床试验 + 专利 + 网络/因果/文本模型的规模化整合，传统人工专家可以做但难以高频、跨疾病、跨资产系统化执行。
- 闭环是否真实：相对真实。Rentosertib、ISM3091、MEN2501 等说明平台输出已经进入 IND/临床/授权，而不是停留在报告层。
- 是否已产资产：是。最高到 Phase IIa 公开数据，多个 Phase I/IND/合作项目。
- 壁垒：中高。平台、模型、实验闭环、管线、BD 网络、上市融资能力组合形成壁垒；单一数据壁垒不可过度夸大。
- 最大风险：Phase IIa 信号无法在更大/更长试验复现；BD 名义总额无法转化为持续现金流；AI 发现靶点的临床成功率未被系统证明高于传统方式。
- 2-3 年价值拐点：
  - Rentosertib 是否进入/完成更大规模后期试验设计与关键读数；
  - Lilly、SK、Menarini、Exelixis 项目是否触发更多临床/监管里程碑；
  - 2026-2028 年收入结构是否从一次性授权/里程碑转向可预测软件 + 重复合作 + 多资产 milestone；
  - 公司是否能证明 31 PCC/13 IND 的“量”转化为可复现的“质”。

## 11. 本技术细分价值定性【降本增效 vs 范式重构】

结论：兼有，当前阶段为“产业验证期”，其中短中期商业价值主要体现为降本增效，长期潜力具备范式重构特征。

降本增效（A）的依据：

- 替代/增强环节：文献综述、组学差异分析、靶点排序、机制证据汇总、专利/临床试验/竞争格局检索、候选适应症和 drug repurposing 初筛。
- 减少实验候选数量：Insilico 披露 PCC 平均 12-18 个月、每项目合成测试 60-200 个分子，相比传统早期发现 2.5-4 年有明显效率主张；但这是公司自述，需要更多第三方项目复现。
- 提高命中率：Rentosertib 证明 AI 关系建模可以产生进入 Phase IIa 的新靶点/新分子组合，但目前样本数不足以证明系统性提高临床成功率。
- 缩短周期：TNIK 项目从 target discovery 到 PCC 约 18 个月、phase 0/1 到 30 个月内，是强案例，但尚不能推广为全行业平均。
- 提前识别风险：PandaOmics 的 novelty、safety、drug presence、commercial tractability、druggability、mechanism clarity 等维度有助于早期过滤；实际能否减少临床失败仍待长期数据。
- 3-5 年商业价值：已形成软件订阅、药企合作、资产授权、milestone 的混合收入模式；Lilly/SK/Sanofi/Exelixis/Menarini 等合作显示药企愿意为该能力付费。

范式重构（B）的依据：

- 传统难点：靶点-疾病-药物关系通常分散在组学、动物模型、临床、专利、文献和专家经验中；跨疾病、跨模态、跨时间的系统性整合成本高、速度慢、偏差大。
- 新一代 AI 新增能力：多组学 + 文本 + 知识图谱 + 生成化学 + 自动化实验闭环，使“从疾病假设到候选分子”的链条可以更高频、可迭代、可复用地运行。
- 已有真实验证：Insilico 的 Rentosertib 是少数同时具备 AI-discovered target、AI-designed molecule、体内外验证、Phase I、Phase IIa 公开结果的案例；Exelixis/Lilly 等 BD 进一步验证药企买单。
- 革命性价值阶段：已不只是理论，但仍未达到成熟产业范式。关键缺口是缺少 Phase III/注册/上市药物证明，缺少跨公司、多项目、可对照的成功率提升数据，也缺少对负数据/失败项目的透明披露。

本细分最终判断：

- 定性：兼有。
- 当前阶段：产业验证期。
- 主要投资含义：本细分在未来 3-5 年更可靠的回报来自“提高早期研发资本效率 + 产生可授权资产 + 支持药企靶点/项目决策”，而不是立即颠覆完整创新药研发流程。真正的范式重构需要看到多个 AI-originated 靶点/分子通过后期临床和监管审批。

一句话总结：Insilico Medicine（英矽智能）是本细分代表公司，最高验证级别为 A（临床注册/IND/合作方BD/上市）+ B（Nature Medicine Phase IIa 论文），投资定位为重点投资级案例但需按上市估值和临床风险持续跟踪；本细分价值定性为兼有，当前处于产业验证期。
