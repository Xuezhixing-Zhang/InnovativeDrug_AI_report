# 商业模式地图

> 口径：按公司当前主要价值捕获方式归类；多模式公司放入最能解释其估值和现金流的主类，并在价值来源中说明混合成分。

| 商业模式 | 代表公司 | 主要收入/价值来源 |
|---|---|---|
| 1. 软件/工具授权 | Schrödinger；CytoReason；Unlearn.AI；QuantHealth；InSilicoTrials；VeriSIM Life；Chemical.AI；BioMap | 软件订阅、模型授权、平台访问费、临床试验设计/模拟服务、药企 R&D 决策工具嵌入。核心风险是价值捕获天花板较低，除非能持续产生专有数据或进入资产权益。 |
| 2. 项目型发现服务 | DP Technology；Galixir；PostEra；Iktos；Chemify；Quris-AI；Mana.bio；Avista；Capsigen；StoneWise | 按项目收费、FTE/研究费、hit/lead 交付、合成路线或验证服务。价值来源偏服务收入，需证明交付能转化为客户 PCC/IND 或保留 IP/数据。 |
| 3. 联合发现 | Isomorphic Labs；Aqemia；Inceptive；Profluent；Dyno；BigHat；LabGenius；Nabla Bio；AI Proteins；Relation；Mirai Bio | 药企提供靶点/数据/疾病领域，公司提供 AI 模型或设计能力，收入来自 upfront、研究费、选择权、里程碑和潜在 royalties。关键是合作项目是否真实推进到候选物。 |
| 4. 联合开发 | XtalPi；MindRank；Earendil/Helixon；Scribe；Arbor；METiS；CorrectSequence；Creyon；Deep Genomics；Owkin | 公司保留部分资产权益或参与后续开发，收入来自 upfront、共同开发分摊、临床/监管里程碑、分成或区域权益。关键是权利归属、数据留存和资金责任。 |
| 5. 平台+自研管线 | Insilico Medicine；Iambic；Recursion；Generate；Absci；AbCellera；Metagenomi；Affinia；4DMT；Cellarity；BenevolentAI；Lantern | 平台对外合作产生现金，自研或控股管线提供 upside；融资和估值依赖管线临床价值。关键是平台是否能重复产生资产，以及失败管线是否反证平台局限。 |
| 6. 自主创新药 | 4DMT；Affinia；Frontera；BioXcel；SOM Biotech；Lantern；Cellarity；MindRank | 主要价值来自自研管线临床推进、适应症扩展和未来商业化。AI是研发手段，不一定是估值核心；需拆分药物本身竞争格局与AI平台贡献。 |
| 7. License-out资产 | Insilico Medicine；Relay Therapeutics；Earendil/Helixon；Generate；XtalPi/DoveTree合作资产；Deep Genomics；SOM Biotech | 通过对外授权、首付款、里程碑和销售分成实现资产价值。质量高于单纯工具合作，但必须区分已收首付款、近端付款和远期 biobucks。 |
| 8. 混合 | XtalPi；Schrödinger；Insilico；Recursion；Generate；Absci；AbCellera；BioMap；Owkin | 同时包含软件/工具、发现服务、合作、管线和授权。优势是现金来源多；风险是叙事复杂，需按业务线拆分毛利、现金消耗、资产权益和真实回款。 |

## 关键判断

- 最可投资的模式通常不是纯软件授权，而是能把平台能力转化为自研资产、合作管线权益或 license-out 现金的混合模式。
- 已披露大额 headline deal 的公司不得直接按总额计入商业验证；只有明确 upfront、已收款、合作方继续推进或候选物进入临床，才上调商业验证分。
- 临床开发模拟、数字孪生和 R&D 软件类公司可作为产业效率标的，但与创新药资产 upside 的映射较弱。
