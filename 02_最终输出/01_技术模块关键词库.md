## 关键词组合公式
> 技术任务词 + 产业化词 + 公司/平台词 + 证据词
> 例：`AI antibody design company partnership pipeline` / `人工智能 抗体设计 公司 平台 管线 合作`

## 通用词包
- 产业化词：company / startup / platform / pipeline / partnership / licensing / biotech / pharma；公司/创业公司/技术平台/药物管线/产业化/药企合作/联合研发/授权/融资
- 资产验证词：development candidate / PCC / IND / clinical trial / Phase 1 / in vivo / experimental validation；候选药物/临床前候选/进入临床/动物实验/体内验证/实验验证
- 商业验证词：upfront / milestone / royalties / co-development / strategic / expanded / repeat partnership；首付款/里程碑/销售分成/联合开发/战略合作/扩大合作/对外授权
- 负面词：failed / discontinued / terminated / layoffs / restructuring / pivot / controversy；临床失败/管线终止/合作终止/裁员/转型/争议/质疑
