## 三、B 技术路线

1. **患者组织多组学 → 靶点/患者分层**：Verge 以 CNS 人脑组织为锚点，整合基因组、蛋白组、单核测序和深临床表型；Owkin 以医院患者网络、病理/空间多组学和临床结局为核心。两者共同优势是更靠近人体疾病，但数据许可、跨中心标准化和可迁移性是关键尽调项。
2. **知识图谱/多源数据融合 → 疾病—基因关系**：BenevolentAI 的 Rosalind 论文（2020-10-26，B 级）用异构图和张量分解预测疾病—基因关系，并进行临床前验证；英矽智能 PandaOmics（2024-05-27，B 级）将多组学、文献和知识网络用于靶点/标志物发现。
3. **患者组织 → 扰动组学 → 转化模型闭环**：Relation 官网（页面元数据初始披露 2023-10-24，C 级）明确描述患者遗传和专有组织组学、扰动组学、转化细胞/临床前模型、高性能实验的闭环，技术链条比纯软件公司更完整。
4. **生命科学基础模型 → 智能实验**：百图生科当前官网公开前端称 xTrimo 跨 DNA、RNA、蛋白和细胞，BioMap OS 连接知识、预测设计、智能实验、数据与训练；目前“20+ PoC”和闭环表现主要为公司自述，需外部论文/客户数据验证。

## 四、C 实验验证

- **BenevolentAI / Rosalind**：2020-10-26 同行评议论文报告对异构图算法预测靶点作临床前验证，属于本模块直接的“模型输出—实验验证”证据；证据为 B 级论文，不等同于药物临床成功。[论文](https://doi.org/10.1038/s41598-020-74922-z)
- **英矽智能 / TNIK**：2024-03-08 在线披露、2025-01 刊期的 Nature Biotechnology 论文报告 INS018_055/ISM001-055 在临床前和临床模型中的数据；2025-06-03 在线发表的 Nature Medicine 随机 Phase 2a 论文进一步提供人体资产验证。靶点识别、分子生成与临床效果需分层评价，不能把整个链条都归因于多组学模型。[临床前/临床模型论文](https://doi.org/10.1038/s41587-024-02143-0)；[Phase 2a 论文](https://doi.org/10.1038/s41591-025-03743-2)
- **Relation**：公司官网称用 perturbational omics、转化细胞和临床前模型测试基因对疾病表型的作用；属于 C 级公司自述，本轮未检出相应同行评议药物项目论文。[科学页](https://www.relationrx.com/science)
- **Owkin / MOSAIC**：2025-05-20 披露的 bioRxiv 预印本报告大规模空间与单细胞多组学肿瘤异质性图谱；可验证数据/分析能力，但不是药物候选的体内疗效验证。[预印本](https://doi.org/10.1101/2025.05.15.654189)
- **百图生科**：官网称已支持 20+ 药物发现、精准医疗和疾病机制 PoC，并形成自动实验反馈；本轮未找到可把具体 PoC 绑定到命名靶点、体内数据或候选药的公开一手证据，暂按 E/C 级。
